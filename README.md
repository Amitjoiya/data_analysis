## ✨ Overview

<img align="right" src="https://media.giphy.com/media/3oKIPEqDGUULpEU0aQ/giphy.gif" width="200"/>

This project dives deep into **Netflix's massive catalog** of movies and TV shows to uncover fascinating insights and trends through beautiful data visualizations.

> 🎯 **Goal**: Transform raw data into meaningful visual stories!

<br clear="right"/>

---

## 📊 Visualizations

<div align="center">

| 🎨 Chart | 📝 Description |
|:--------:|:---------------|
| 🎬 **Type Distribution** | Bar chart showing the split between Movies and TV Shows |
| ⭐ **Rating Distribution** | Pie chart displaying content ratings breakdown |
| ⏱️ **Movie Duration** | Histogram of movie lengths in minutes |
| 📅 **Release Year Trends** | Scatter plot of titles released per year |
| 🌍 **Top 10 Countries** | Bar chart of countries with most content |
| 🎥 **Top 10 Directors** | Bar chart of most prolific directors |

</div>
